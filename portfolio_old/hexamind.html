guidinger design

hexamind app

overview

This design is for a native application. It is specifically designed for the iOS of an iPad in mind. The design language loosely adheres to the UI structure of an iOS application. This application's function is to provide a high technology approach to mindmapping. Mindmapping is the practice of "mapping" thoughts that are connected. The central design style is based around the shape of a hexagon. Functionally, nodes in the shape of joined hexagons are locked into a hexagonal grid and can be edited, and moved around the grid with streamlined gestural commands. This design uses outer glow and gradients to create depth. The intuitive interface leaves some room to explore in a contrast rich environment. It is designed to spark creativity and ideation.

challenge

This concept was unique in that I was designing for a different platform than web. While the concepts could be transferred to web technologies, the initial iteration was meant for a native application environment. This removed any form of coding and programming from my end. However, considerations for developing the app were extremely important in this design, as core functionality is key. Creating for an app requires forethought of how development and deployment will happen.

approach

I initially approached this concept with many features in mind. However, after researching other similar functional applications, I realized that an over-featured application would unnecessarily complicate UX and that it could initially drive users away. I stripped the concept down to its core components. I was left with a "home" page, an "archive" or "open" page, a settings page, and the actual mind map.

conclusion

I really enjoyed the change of gears to be able to focus solely on design and user interactions. This project also relied heavily on prototypes that were created in Adobe XD. This application was intentionally kept to a small scope so as to focus its intial design on core function. This grew my understanding considerations of app development and user experience throughout the design process.
