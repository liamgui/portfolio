panic button app

view app

overview

This concept is a web application that helps users in moments of panic. It consists of a panic button that randomly redirects the user to one of several pages. The Panic Button app is fully functional. The pages are designed to help relieve stress by providing calming sensory input, such as ocean waves, calming music, restful quotes, atmospheric graphics, cathartic interactivity, and harmonious visual balance. The interface is simple and straightforward, allowing users to simply experience the benefits that the panic button app provides.

challenge

This application required both offline and online support. Offline support was created through the use of service workers.

approach

I began by researching soothing elements, methods of panic reduction, and brainstorming multiple ways to implement these elements and methods. I recognized that the user flow needed to be a smooth but quick transition from the panic button to a randomly selected relaxation page, and therefore the panic button itself was carefully designed to not overwhelm the user. I utilized the HTML canvas element as well as JavaScript to guide the user in exercises of a calming nature.

conclusion

The Panic App has the unique ability to quickly and effectively achieve user goals, while maintaining reusability across various situational needs. Using web app technologies, it can be used offline as well as online for more features. The pages can be added to and changed without affecting the user flow.
